# MERN Stack Todo app

This is a front end part (React js) of the full stack Todo app built with the MERN stack.

## Setup the project

Make sure you have pnpm an mongo db compass installed in your system.

```bash
pnpm i
```

**Run the frontend:**

```bash
pnpm dev
```

Go to <http://localhost:5173/>
